# Student Attendance Management System

A desktop-based Java application for managing student attendance in educational institutions.

## Technologies Used
- Java (JDK 8 or above)
- Java Swing (GUI)
- JDBC (Database Connectivity)
- MySQL Database
- Eclipse IDE

## Features
- User Login (Admin / Teacher)
- Student Registration
- Mark Daily Attendance (Present / Absent)
- View Attendance Records
- Search Student by ID or Roll Number
- Student Profile with Photo Upload
- Report Generation

## Database
- Database Name: attendance_db
- Tables: students, attendance, users

## How to Run
1. Install Java JDK and Eclipse IDE
2. Install MySQL and create database attendance_db
3. Run the SQL file to create tables
4. Open project in Eclipse
5. Run LoginFrame.java
